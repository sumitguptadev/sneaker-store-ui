# 👟 Sneaker Store UI

A stylish and responsive **Sneaker Store website** built using **HTML and CSS**.
This project showcases a collection of Air Jordan sneakers with modern UI design and product cards.

---

## 🚀 Features

* 🧾 Clean and modern UI design
* 👟 Product cards with images, price, and details
* 🛒 Shopping cart icon layout
* 📱 Responsive structure (basic)
* 🎨 Custom fonts and styling using CSS

---

## 📂 Project Structure

